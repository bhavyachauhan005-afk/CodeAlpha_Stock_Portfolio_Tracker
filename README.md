# CodeAlpha Stock Portfolio Tracker

## Description
This is a simple Stock Portfolio Tracker developed in Python as part of the CodeAlpha Python Programming Internship.

## Features
- Tracks stocks and quantities
- Uses predefined stock prices
- Calculates total investment value
- Saves portfolio summary to a text file

## Technologies Used
- Python 3

## How to Run
1. Download the project.
2. Open the project in VS Code.
3. Run:
   ```bash
   python stock_portfolio_tracker.py
   ```
